# DealDrop API

This service will host the modular monolith described in the architecture plan.

## Planned module boundaries

- Identity
- Discovery
- Listings
- Collections
- Search
- Saved
- Contributions
- Moderation
- Trust
- Karma
- Notifications
- Admin Ops

The contract-first source of truth currently lives in [`packages/contracts/openapi/dealdrop.v1.yaml`](../../packages/contracts/openapi/dealdrop.v1.yaml).
