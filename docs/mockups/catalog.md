# Mockup Catalog

The citywide DealDrop PNGs are the binding visual source of truth. Bulletin and campus-branded screens are reference-only where the canonical citywide family does not yet include a required flow.

| Screen | Source Family | Authority | Visible Components | Implied Actions | Notes |
| --- | --- | --- | --- | --- | --- |

| Sign-up form | Citywide DealDrop | Canonical | back button, location, username, age, email, large CTA | submit account, return | Use citywide spacing and rounded field shells |
| Deals feed | Citywide DealDrop | Canonical | location row, search, filter chips, stacked offer cards, save and overflow actions, bottom nav | search, filter, open detail, save | Core feed composition reference |
| Map | Citywide DealDrop | Canonical | large map surface, floating pin cards, zoom controls, bottom preview card, bottom nav | tap pin, zoom, open preview detail | Use as composition reference even before SDK integration |
| Karma / points | Citywide DealDrop | Canonical | large balance card, progress, badges, leaderboard card, bottom nav | inspect status, open leaderboard | Reference for gamification hierarchy |
| Profile | Citywide DealDrop | Canonical | user summary, friends list, add friends row, support rows, bottom nav | view account, open support/share | Profile is secondary stack, not a bottom tab in final IA |
